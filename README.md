# Seckill
## 简介

通过SpringBoot快速搭建前后端分离的电商基础秒杀项目。在秒杀基础项目中完成用户otp注册、登录、查看、商品列表、进入商品详情以及倒计时秒杀下单的基本流程。再基于基础秒杀项目，优化并实现高性能的秒杀系统。

## 开发环境

### 后端

- JDK1.8
- Maven3
- SpringBoot2.0.5
- MySQL5.6
- Mybatis

### 前端

- Metronic模板

> [Metronic 6.0](http://metronic.kp7.cn/) 是一套精美的响应式后台管理模板，基于强大的 Twitter Bootstrap 3.3.4 框架实现。
